# 中文說明
傳送玩家到其他玩家身上或準心上

* 原理
	* 管理員輸入!teleport 可以傳送指定玩家到準心上或是其他玩家身上

* 功能
	* 可以加入到管理員菜單下，輸入!admin->玩家指令->傳送玩家
